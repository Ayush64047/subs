# Subtitle-API

A subtitle application user interface ( API )

---

## Usage

### For results

```
https://subtitle.api.fayas.me/results/?query=<your-query>
```

### For specific result

```
https://subtitle.api.fayas.me/result/?query=<your-keyword>
```
